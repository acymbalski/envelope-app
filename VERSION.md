# Version History

## v1.0.0 - Initial Release (December 18, 2024)

### Features
- 5 Christmas gift envelopes with deterministic ordering
- One gift per day limit with override option
- Red/white Christmas theme (colorblind-friendly)
- Mobile and desktop responsive design
- localStorage persistence for opened gifts
- Timezone-aware daily reset
- Subtle sound effects on envelope opening
- Falling snowflake animations
- Gift images: 5 photos pre-configured

### Technical Stack
- Pure HTML/CSS/JavaScript (no dependencies)
- localStorage for state management
- Web Audio API for sound (graceful degradation)
- CSS Grid for responsive layout
- CSS transforms for animations

### Deployment Target
- Domain: xmas.cymbal.ski
- Hosting: TBD (GitHub Pages, Unraid, or Netlify)
- User timezone: New Zealand

### Files
- index.html (Main page)
- styles.css (Styling)
- app.js (Logic)
- gifts-list.json (Gift configuration)
- gifts/ (5 images)
- Documentation (README, QUICKSTART, DEPLOYMENT, TESTING, SUMMARY)
